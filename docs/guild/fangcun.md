# 方寸山三星洞

灵台方寸山开山祖师是菩提祖师，传说他是佛的十大弟子之一，又是一个道佛结合的人物。
方寸山门规严谨，要求弟子做事端庄。武功方面初进极易，而后对悟性要求极高。其方寸，灵台，及菩提祖师的法号斜月三星都是心的隐意。
